# 🌐 Personal Portfolio Website

![Portfolio Banner](https://img.shields.io/badge/Portfolio-Website-blue?style=for-the-badge\&logo=google-chrome)

A modern **personal portfolio website** built to showcase my **skills, projects, and experience** in web development and data analytics.
This portfolio highlights my technical abilities, projects, and learning journey in an interactive and visually appealing way.

---

# 🚀 About the Project

This portfolio website is designed to present my **professional profile, technical skills, and projects** in one place.

The website uses **Django for backend functionality** while the frontend is built using **HTML, CSS, and JavaScript** with modern UI effects including **3D visual effects and smooth animations**.

The goal of this project is to create a **dynamic, responsive, and visually engaging portfolio** that reflects my development skills.
